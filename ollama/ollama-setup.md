## 💾 Install
1. Install the Ollama client per platform instructions (see official site).  

## 📥 Pull Models
2. Pull or load the desired models into Ollama:
```bash
ollama pull <model-name>
```

## ▶️ Serve Locally
3. Serve models locally with Ollama (example):
```bash
ollama run <model-name>
```
